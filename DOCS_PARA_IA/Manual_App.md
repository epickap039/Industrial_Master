# Manual de Usuario - App Industrial Master

## Módulos Principales

### 1. Catálogo Maestro

- Aquí puedes ver todas las piezas que el sistema ha "aprendido".
- **Editar:** Usa el icono de lápiz para corregir una descripción o proceso. Estos cambios se verán reflejados en el auto-rellenado de futuras listas.

### 2. Auditoría de Conflictos

- Muestra piezas que se importaron desde un Excel con datos que NO coinciden con el Maestro.
- **Acción "Hacer Oficial":** Si determinas que el dato del nuevo Excel es el correcto, presiona este botón para que el Maestro se actualice automáticamente con esa información.

### 3. Creador de Listas

- Escribe el código y presiona Enter.
- **Borde Naranja:** Significa que la pieza es nueva y no está en el catálogo. Debes llenarla manualmente.
- **Auto-relleno:** Si la pieza existe, el sistema completará los campos por ti.
